Errores de Sofware

Un error de software o simplemente fallo, también conocido por el inglés como bug, es un problema en un programa de computador o sistema de software que desencadena un resultado indeseado. Los programas que ayudan a la detección y eliminación de errores de software son denominados depuradores (en inglés, debuggers).


A continuacion añadiremos algunos errores mas reconocidos que se presententa dia a dia en nuestras computadoras, quizá encuentres una solucion para ti.
